---
title: "MRound"
---


Rondt het getal naar beneden af naar het dichtstbijzijnde veelvoud van het opgegeven getal

**Categorie:** `math`

**Syntax:**
```
MRound([getal], veelvoud)
```

**Retourneert:** `Number`

**Contextfiltering:** ✓ Ja

## Parameters

| Naam | Type | Vereist | Beschrijving |
|------|------|---------|--------------|
| `getal` | `getal of kolom` | ✓ Ja | Het getal dat wordt afgerond |
| `veelvoud` | `getal` | ✗ Nee | Het veelvoud waarnaar het getal wordt afgerond |

**Toegestane kolomtypen voor `getal`:** INT, FLOAT, DECIMAL

## Validatie
- Minimum parameters: 1
- Maximum parameters: 2

## Voorbeelden

```
MRound([Getal])
```
Rondt de kolom Getal af met een factor van 1 omdat er geen argument is opgegeven. Dit is equivalent aan het resultaat van de factor (-1).

```
MRound([Kosten], 5)
```
Rondt elke waarde in de kolom Kosten af naar het dichtstbijzijnde veelvoud van 5

```
MRound(23, 5)
```
Retourneert 25
**Resultaat:** `25`

```
MRound(27, 5)
```
Retourneert 25
**Resultaat:** `25`

```
MRound(23, -5)
```
Retourneert 25 (negatief factorteken wordt genegeerd)
**Resultaat:** `25`
