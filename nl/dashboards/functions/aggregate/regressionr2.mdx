---
title: "RegressionR2"
---


Determinatiecoëfficiënt (R-kwadraat) van lineaire regressie

**Categorie:** `aggregate`

**Syntax:**
```
RegressionR2([y], [x])
```

**Retourneert:** `Float`

**Contextfiltering:** ✗ Nee

## Parameters

| Naam | Type | Vereist | Beschrijving |
|------|------|---------|--------------|
| `y` | `kolom` | ✓ Ja |  |
| `x` | `kolom` | ✓ Ja |  |

**Toegestane kolomtypen voor `y`:** INT, FLOAT, DECIMAL

**Toegestane kolomtypen voor `x`:** INT, FLOAT, DECIMAL

## Validatie
- Minimum parameters: 2
- Maximum parameters: 2

## Voorbeelden

```
RegressionR2([Y], [X])
```
