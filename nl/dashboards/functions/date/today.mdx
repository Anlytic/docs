---
title: "Today"
---


Retourneert de huidige datum in ISO-formaat met behulp van de tijdzone van het account.

**Categorie:** `date`

**Syntax:**
```
Today()
```

**Retourneert:** `DateColumn`

**Contextfiltering:** ✗ Nee

## Parameters

Geen

## Validatie
- Minimum parameters: 0
- Maximum parameters: 0

## Voorbeelden

```
Today()
```
Retourneert de huidige datum.
