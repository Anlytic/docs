---
title: "Now"
---


Retourneert de huidige datum en tijd met behulp van de tijdzone van je organisatieaccount.

**Categorie:** `date`

**Syntax:**
```
Now()
```

**Retourneert:** `DateTimeColumn`

**Contextfiltering:** ✗ Nee

## Parameters

Geen

## Validatie
- Minimum parameters: 0
- Maximum parameters: 0

## Voorbeelden

```
Now()
```
Retourneert de huidige datum en tijd.
