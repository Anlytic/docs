---
title: "InDateRange"
---


Bepaalt of een datum binnen een opgegeven datumbereik valt.

**Categorie:** `date`

**Syntax:**
```
InDateRange([datum_kolom], "richting", "periode", |lengte|, |offset|, |vandaag|)
```

**Retourneert:** `BooleanColumn`

**Contextfiltering:** ✓ Ja

## Parameters

| Naam | Type | Vereist | Beschrijving |
|------|------|---------|--------------|
| `datum_kolom` | `kolom\|datetime` | ✓ Ja | De datum die wordt geëvalueerd. |
| `richting` | `text` | ✓ Ja | Richting om de datumbereik-offset toe te passen. |
| `periode` | `text` | ✓ Ja | Grootte van het datumvenster. |
| `lengte` | `getal` | ✗ Nee | Aantal perioden dat in het bereik wordt opgenomen. Standaard is 1. |
| `offset` | `getal` | ✗ Nee | Aantal perioden om het begin van het bereik te verschuiven. Standaard is 0. |
| `vandaag` | `kolom\|datetime` | ✗ Nee | Vervangende waarde voor Vandaag bij het berekenen ten opzichte van de huidige datum. |

**Toegestane waarden voor `richting`:** `last`, `next`, `current`, `to_date`

**Toegestane waarden voor `periode`:** `year`, `quarter`, `month`, `week`, `day`, `hour`, `minute`, `second`

**Toegestane kolomtypen:** DATETIME, DATE

## Validatie
- Minimum parameters: 3
- Maximum parameters: 6

## Voorbeelden

```
InDateRange([Datum], "current", "month")
```
Retourneert Waar voor alle datums in de huidige maand.

```
InDateRange([Datum], "last", "week", 3)
```
Retourneert Waar voor alle datums in de afgelopen drie weken.

```
InDateRange([Datum], "last", "week", 1, 3)
```
Retourneert Waar voor alle datums in de week die drie weken geleden begon.

```
InDateRange([Datum], "to_date", "year", 1, -1)
```
Retourneert Waar voor alle datums binnen het afgelopen jaar tot aan de datum van vandaag.
