# Today

Returns the current date in ISO format using the Account Time Zone.

**Category:** `date`

**Syntax:**
```
Today()
```

**Returns:** `DateColumn`

**Context Filtering:** ✗ No

## Parameters

None

## Validation
- Minimum parameters: 0
- Maximum parameters: 0

## Examples

```
Today()
```
Returns the current date.
