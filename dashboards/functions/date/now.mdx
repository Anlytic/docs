# Now

Returns the current date and time using your organization's account timezone.

**Category:** `date`

**Syntax:**
```
Now()
```

**Returns:** `DateTimeColumn`

**Context Filtering:** ✗ No

## Parameters

None

## Validation
- Minimum parameters: 0
- Maximum parameters: 0

## Examples

```
Now()
```
Returns current date and time.
