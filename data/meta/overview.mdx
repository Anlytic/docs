## What does meta data do?

Metadata plays an important role in a data catalog by providing essential information about the data that you need, to create visualizations, conduct analysis, and create  reports effectively. It ensures data trustworthiness, enables data exploration, and helps you to make informed decision based on accurate and reliable data.

Metadata serves as essential information about the data, including tables and columns source names and descriptions about the data. You can see metadata by hovering over the '?' on the table or column names in the catalog while creating visualizations.
