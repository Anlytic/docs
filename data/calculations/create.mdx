# Calculate

In this page we are going to look at how we can create metrics and calculated columns in Anlytic application. You can create these formulas when you are setting up your data connection as well as in the connections which are already established. To create metrics and calculated columns for the connection already established go to the edit metrics and calculated column page.

## How to create Metrics?

While creating a new connection, right after the step of creating data relationships comes the calculate step. In this step you can create metrics formulas.

1. Give a name to your metric.
2. Select the data type.
3. Select tables from the dropdown.
4. Select the reference table(s). You can select multiple reference tables.
5. Add a description of the metric.
6. Enter SQL formula and click on save.

**The metric is saved and ready to use.**

## How to create calculated columns?

To create a calculated column, you can define an SQL expression that combines existing columns or applies specific transformations. These expressions are evaluated when the query is run, producing new columns in your dataset that can be used in visualizations and reports.
<br></br>

Follow these steps to create a calculated column:

1. Give a name to your calculated column.
2. Select the data type.
3. Select tables from the dropdown.
4. Select reference table(s). You can select multiple reference tables.
5. Add a description of the calculated column.
6. Enter SQL formula and click on save.

**The calculated column is created and saved.**

![alt text](/Resources/create%20calculated%20columns.gif)
