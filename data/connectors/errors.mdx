title: "Error handling"
---


In this chapter we are going to walk you thorugh how you can handle errors during your data connection.

What happens if something goes wrong during data connection setup? Don't worry—our data connectors are equipped with powerful error handling mechanisms. If there's an error in the connection setup process, our connectors are designed to handle the errors. This means you'll receive clear error messages, allowing you to quickly diagnose and resolve any issues.

When the connection error occurs, you will see a connection failed message.

![(Image of the file connection failed)](/Resources/Connector%20test%20error%201.png)

You can click on the arrow button on the error message, to expand the error message to see where the error occured. 

![(image of expanded error message)](/Resources/Connector%20test%20error%202%20expanded.png)

You can also expand the error message pannel to full display to go through all the error messages to resolve the error. 

![(image of the fully expanded error message box)](/Resources/Connector%20test%20error%203%20(log%20expanded).png)

If you think there is a bug for which the error is occuring, use the bug report option by clicking on the '?' at the bottom left corner of your screen. 

![(image of set up connection page with bug report button highlighted)](/Resources/bug%20report.png)
