# Data connectors

Data connectors allows you to access and utilize your data for analysis and reporting purposes through Anlytic platform.

![(image of data connector page)](/Resources/Connector%20-%20select%20connection.png)

Imagine you have data scattered across various databases, or in excel files. Our data connectors allows you to connect the data to the platform, prepare the data and finally organize the data into a catalog. Therefore, easliy utilize the data to create visualizations that can give you useful insights allowing a user to make informed decisions. 

In this chapter we are going to walk you through how you can setup data connections, handle errors, add meta data, create relationships between data, create fromulas, and finally organize data into a catalog.
