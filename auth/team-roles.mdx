title: "Team roles"
---

On this page we will walk you through team roles and permissions related to the roles. There are four team-level roles available in the Anlytic platform. They are:

1. Team Owner
2. Team maintainer
3. Team editor
4. Team guest

## Team Owner

As a team owner  you can perform the following tasks:

1. Appoint owner status.
2. Appoint maintainer status.
3. Appoint editor status.
4. Appoint guest status.
5. Remove user from a team.
6. add users to a team. 
7. Edit dashboards.
8. Create dashboards.
9. View dashboards


## Team Maintainer

As a team maintainer you can perform the following tasks:

1. Appoint maintainer status.
2. Appoint editor status.
3. Appoint guest status.
4. Remove user from a team.
5. Add users to a team. 
6. Edit dashboards.
7. Create dashboards.
8. View dashboards

## Team Editor

As a team editor, you can perform the following tasks: 

1. Edit dashboards
2. View dashboards

## Team Guest

As a team guest, you can perform the following tasks:

1. View dashboards.
